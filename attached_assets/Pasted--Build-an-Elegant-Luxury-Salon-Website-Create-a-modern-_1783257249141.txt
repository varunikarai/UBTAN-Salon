# Build an Elegant Luxury Salon Website

Create a modern, responsive, single-page website using **HTML, CSS, and vanilla JavaScript only** (no backend, no paid APIs, no frameworks). The website should be lightweight and work perfectly on Replit's free plan.

## Brand Details

**Salon Name:** UBTAN

**Owner:** Neelu Rai

**Tagline:** "Blossom into a new u..."

**Business:** Ladies-only premium beauty salon.

The overall feeling should be:

* Elegant
* Premium
* Luxurious
* Feminine
* Warm
* Sophisticated
* Minimalistic

Avoid flashy animations or clutter.

---

## Color Palette

Use a luxury combination of:

* Matte Black (#111111)
* Rich Gold (#D4AF37)
* Champagne Gold (#E5C97A)
* Warm Beige (#F5EFE6)
* Ivory (#FCFAF7)
* White text where appropriate

The website should feel like entering an expensive boutique salon.

---

## Animations

Use tasteful animations only.

## Responsiveness

Must look perfect on:

* Mobile
* Tablet
* Laptop
* Desktop

Use Flexbox and CSS Grid.

---

## Extra Luxury Details

Add:

* thin gold dividers
* rounded elegant cards
* subtle shadows
* glassmorphism where appropriate
* soft gradients
* gold accent lines
* smooth transitions
* custom scrollbar in gold theme

---

## Booking Button

The "Book Appointment" button should open a stylish modal where users can enter:

* Name
* Phone
* Service
* Preferred Date
* Message

No backend required.

On submit, simply show:

"Thank you! We'll contact you shortly."

---

## Code Quality

Separate files:

* index.html
* style.css
* script.js

Write clean, well-commented code.

No external libraries except Google Fonts.

Keep the website lightweight and fast.

---

## Overall Goal

The final website should feel like the online presence of a premium luxury salon, with an elegant black, gold, and beige aesthetic that inspires trust, sophistication, and comfort. Every section should have polished spacing, refined typography, smooth interactions, and a high-end visual identity worthy of a boutique beauty brand.
